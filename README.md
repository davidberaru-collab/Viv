# Apostolic Alignment School of Ministry Website

Next.js + TypeScript + Tailwind CSS website for Christian leadership training and speaking.

## Run locally

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start dev server:
   ```bash
   npm run dev
   ```
3. Open http://localhost:3000

## Notes
- Built with Next.js App Router.
- Placeholder content and image blocks are included for easy editing.
- Comments are included in code where future CMS/CRM integrations can be connected.
